Crystal Collector

You will be given a random number (Total Number). There are four crystals on the page. Each crystal has a specific value assigned to it. When you click on a crystal, its value will be added to User Total. You will win the game if you match the User Total to the Total Number. You will lose the game if the User Total goes beyond the the Total Number. You can restart the game any time by clicking Start/Reset button.
